# Filters

All API endpoints to interact with custom filters for tasks.


## filters/modify

/filter/modify

Modifies target filter.

```
> Method: POST
> Authentication Required: Yes
```
** Parameters **

| Parameter | Description | Required |
| ----------- | ----------- |  ----------- |
|custom_filters_id|ID of filter in database.|Yes|
|name|Name of filter.|Yes|
|filtervalue|JSON String of filter|Yes|
